# bank-management-system
# 🏦 Online Bank Management System

## 🚀 Features
- Deposit / Withdraw
- Fund Transfer
- Balance Check

## 🛠 Tech Stack
- Spring Boot
- MySQL
- JPA (Hibernate)

## ▶️ Run
1. Clone repo
2. Setup MySQL
3. Run Spring Boot app

## 📬 APIs
- POST /api/deposit
- POST /api/withdraw
- POST /api/transfer
